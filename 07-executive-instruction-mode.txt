EXECUTIVE INSTRUCTION MODE

This is your operator model applied to interaction style. Your profile defines how you communicate — this chapter is about encoding that into specific behavioral rules your agents follow. By default, most AI tools are annoyingly consultative — they ask questions, propose options, explain their reasoning, and end every message with "let me know if you need anything else." Executive instruction mode turns that off.

THE DEFAULT YOU'RE REPLACING

Out of the box, AI agents:
  Ask clarifying questions before doing anything
  Offer multiple approaches for you to choose from
  Explain their reasoning step by step
  Suggest related improvements you didn't ask for
  End with "would you like me to also..."

For some people this is fine. For others — especially if you have ADHD or are neurodivergent — it's actively harmful. Every unsolicited question is a focus interrupt. Every suggestion menu is cognitive load dumped back onto you. You hired the assistant to carry the weight, not hand it back.

HOW TO SET IT UP

Add an interaction style section to your global AGENTS.md. Here's a starting point — adjust to fit how you actually want to work:

  Interpret my intent, not just my words. If I say "fix the auth flow," investigate and fix it. Don't ask me which part.
  Report what you did, not how you got there. "Fixed the token refresh bug" — not a five-paragraph debugging narrative.
  Make small decisions silently. Naming, file placement, import order — just pick something reasonable.
  No suggestion menus. Don't offer me options I didn't ask for.
  No closing offers. Don't end with "let me know if you need anything else."
  No scope expansion. Don't mention related improvements or follow-on tasks unless I ask.
  One thread at a time.

WHEN THE AGENT SHOULD PAUSE

There are still times an agent should check with you:
  Your instruction is genuinely ambiguous and both interpretations have big consequences
  The action is irreversible (deleting data, force-pushing, modifying production)
  The agent is stuck and can't make progress without your input

In those cases: one clear question, get the answer, keep going. Not a conversation.

MAKING IT YOUR OWN

Executive instruction mode is a starting point. Maybe you want more explanation. Maybe you want less. Maybe you want terse output for code tasks and detailed output for research. The point is: you define it in a file, and every agent reads it. You stop having to correct the same habits session after session.

The communication profiling chapter (09) takes this further — your agent can analyze how you actually communicate and generate this section for you.
